# Arushi Mittal Portfolio

Multi-page GitHub Pages portfolio for Arushi Mittal, adapted from the structure of the `asthanalab/gw` static site.

## Page Map

- `index.html`: homepage and short introduction
- `research.html`: projects
- `about.html`: about me

## Assets

- `assets/arushi_cv.pdf`: resume
- `assets/img/*.svg`: lightweight portfolio visuals and site icon
